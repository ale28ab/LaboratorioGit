# Laboratorio Git — Trabajo en Equipo

Este repositorio contiene la práctica colaborativa de comandos **Git**, realizada en conjunto para comprender el flujo completo de trabajo con control de versiones, desde la creación del repositorio hasta la modificación, fusión e historial de cambios.

---

## Comandos Explicados

### 1️⃣ `git add`
Agrega archivos al área de preparación (*staging area*), indicando qué cambios serán incluidos en el próximo commit.

git add README.md
git add .
